# 클라이언트 공유용 — 화면 그대로 저장하기

브라우저 **인쇄 → PDF**는 배경·레이아웃이 달라질 수 있습니다.  
아래 방법은 **지금 보는 화면과 동일하게** PNG·PDF를 만듭니다.

## 한 번만 설치

`seah` 폴더에서 터미널 실행:

```bash
cd seah
npm install
npx playwright install chromium
```

## 캡처 실행

```bash
npm run capture
```

## 결과 파일

`seah/exports/` 폴더에 생성됩니다.

- `글로벌세아-메인-YYYY-MM-DD.png` — 클라이언트 전달용 (권장)
- `글로벌세아-메인-YYYY-MM-DD.pdf` — 같은 화면 기준 PDF

캡처 해상도: 가로 **1280px** (데스크톱 화면 기준), 고해상도(2x).

> **2026-06-13:** 히어로·서브배너·사업·실적 이미지는 대부분 `assets/images/` 로컬 파일 사용. 캡처 시 인터넷 없이도 대부분 정상 표시.

## 수동으로 빠르게 (설치 없이)

1. Chrome에서 `index.html` 열기
2. **F12** → `Ctrl+Shift+P` → `Capture full size screenshot` 입력 후 실행  
3. 저장된 PNG를 카카오톡·메일로 전송

PC 미리보기 모드(`responsive: false`, 1200px) 기준으로 캡처하면 클라이언트 검수용 화면과 동일합니다.
