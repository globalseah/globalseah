## 2) 2026-06-19 라운드 — 콘텐츠·게시판·문의

### 실적현황 (`js/portfolio-data.js`)
- 클라이언트 실데이터 **45건** (`assets/images/portfolio/01~46.png`, 44번 삭제)
- 목록 **5×2**, **10건 페이지네이션**, 최신순 역순
- 메인 마퀴 **최근 15건**, 애니메이션 **90초**
- 호버: 이름 → 용도 → 위치 → **사업내용** (하단)

### 공지사항 (`js/notice-data.js`)
- **6건** 이미지 게시 (`assets/images/notice/`)
- 상세 `notice/view.html?id=N`, 이전/다음글, **목록으로**(우측 하단)
- 목록 **10건 단위 페이지네이션** (`js/board-pagination.js`)

### 채용현황 (`js/recruit-data.js`)
- **16건** A안(원문 텍스트) — 상세 `notice/recruit/view.html?id=N`
- 마감: 제목 뒤 `[마감]` (예: 김포통진읍)
- 담당자 연락처 `tel:` 링크, 목록 **10건 페이지네이션**
- 등록일: 1~2일 간격 (최신 6/18 = 인선서구 청라병원)

### 문의 폼 (`contact/`)
- 연락처·이메일 **둘 중 하나 필수**
- **사업부문** 드롭다운 필수 (5종)
- 클라이언트·서버 검증 동기화

### 사이트명·메타
- head/OG: **글로벌세아종합관리** (본문 회사명 `(주)글로벌세아` 유지)
- 전 페이지 `title` · `description` · `canonical` · `og:*` 통일

### 사업부문 명칭
- `미화관리` → **미화·방역 관리** (GNB·메인·메타·cleaning 페이지)
