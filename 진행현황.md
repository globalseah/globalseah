- **Resend 도메인** `globalseah.com` 가비아 DNS 등록 → **Verified**
- 발신: `contact@globalseah.com` (운영)

### PC 고정 레이아웃 (2단계)
- 배경(헤더·히어로·푸터) **화면 전체 너비**, 콘텐츠만 inner **1280px** 중앙정렬
- `html.desktop-only` 브레이크포인트 격리
  - `min-width` 쿼리 10곳 → PC 모드에서 **항상 PC 배치** 유지
  - `max-width` 쿼리 14곳 → `html:not(.desktop-only)`로 한정 (창 축소 시 레이아웃 무너짐 방지)
- `responsive: true` 전환 시 기존 반응형 코드 그대로 복귀 가능
