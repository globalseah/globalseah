## 1) 이번 라운드 반영 완료

- 메인 헤더/GNB 개편
  - 메인에서 투명 헤더 + 호버 시 가로 서브바
  - 서브바는 메뉴별로 개별 노출 (회사소개/사업부문/공지 및 채용현황)
  - 스크롤 고정 헤더 제거
- PC 고정 레이아웃 1단계 적용
  - `responsive: false`
  - 창 너비 축소 시 레이아웃 변형 대신 가로 스크롤 유지
  - `desktop-only` 기준으로 PC 강제 스타일/모바일 쿼리 분기 처리
- 메인 구조/카피 변경
  - 섹션 순서: 히어로 → 사업부문 → 공지·채용·문의 → 실적현황
  - 히어로 카피 갱신:
    - `Better Environment, Better Tomorrow`
    - `사람과 공간을 연결하는 종합관리 전문기업`
    - `더 쾌적한 환경, 더 나은 내일` (보조문구 스타일로 복원)
  - 상단 영문 폰트: `EB Garamond`(italic)
- 메인 실적현황 동작 변경
  - 카드형 고정 그리드 → 에스컬레이터식 무한 흐름 마퀴
  - 마우스오버 시 정지
- 메인 하단 3열 구성
  - 공지사항 + 채용현황 + 문의배너
  - 공지/채용 카드 라운드 적용 및 문의배너와 높이 맞춤
- 공지 영역 확장
  - 메뉴명: `공지사항` → `공지 및 채용현황`
  - `notice/recruit.html` 신규 생성
  - 공지/채용 좌측 탭 구조 적용 (회사소개와 동일 패턴)
- 인증현황 페이지 개편
  - 10칸 썸네일 그리드(정사각형)로 재구성
  - 이미지 미보유 항목은 `이미지 준비중` 표시
  - 반영 이미지: 사업자등록증, 근로자파견사업 허가증, 건물위생관리업 신고증, 분사기 소지허가증

  - `/` → `>`
