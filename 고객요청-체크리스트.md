### A-5. 네이버 연동 (지도 API · 서치어드바이저)

> **메모 — 고객사 요청 필수**  
> 네이버 지도 API(NCP Maps)와 네이버 서치어드바이저 설정은 **네이버 계정 로그인**이 필요합니다.  
> 개발사가 대신 신청·연동하려면 고객사에 **네이버 ID / PW 전달**을 요청하거나,  
> 고객사 담당자가 NCP·서치어드바이저에 직접 가입 후 **Client ID·인증 파일·사이트 등록** 결과만 전달하는 방식 중 하나로 진행합니다.

#### 네이버 지도 API (찾아오시는길)

- [ ] **네이버 ID / PW** (또는 NCP 콘솔 접근 권한) — 개발사 대행 시
- [ ] 네이버 클라우드 플랫폼(NCP) **Application 등록** (Maps 사용 신청)
- [ ] **Client ID** (웹 Dynamic Map용) 발급 후 전달
- [ ] 서비스 URL 등록: 로컬(`localhost`/`127.0.0.1`) + **운영 도메인** (배포 URL 확정 후)
- [ ] 찾아오시는길 **표시 좌표**(위도·경도) 또는 네이버지도 공유 링크
- [ ] (선택) 대중교통·주차 안내 문구

#### 네이버 서치어드바이저 (검색 노출·사이트 등록)

- [ ] **네이버 ID / PW** (또는 서치어드바이저 사이트 소유 확인용 계정) — 개발사 대행 시
- [ ] 서치어드바이저 **사이트 등록** (운영 도메인 확정 후)
- [ ] **사이트 소유 확인** (HTML 메타 태그 또는 파일 업로드 방식 중 선택)
- [ ] **RSS** 또는 **사이트맵(sitemap.xml)** URL (공지 등 갱신 페이지 연동 시)
- [ ] robots.txt · 수집 요청 정책 (필요 시)
