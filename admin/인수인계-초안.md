# 글로벌세아 관리자 — 인수인계 초안 (Phase 8-9)

> 배포·계정 확정 후 대표자에게 전달할 내용 초안입니다. 최종 URL·계정은 배포 시 채워 넣으세요.

## 접속 방법

| 항목 | 내용 |
|------|------|
| 관리자 주소 | `https://globalseah.com/admin/` (끝 슬래시 포함) |
| 로그인 | 브라우저 Basic Auth 팝업 — ID / 비밀번호 (별도 전달) |
| 공개 사이트 | `https://globalseah.com` |

## 메뉴 안내

| 메뉴 | 용도 |
|------|------|
| 공지사항 | 텍스트·이미지(복수) 공지 등록 |
| 실적현황 | 시설 사진 + 시설명·용도·위치·사업내용 |
| 채용현황 | 8항목 구조화 채용 공고 + 담당자 연락처 |
| 방문 통계 | GA4 방문자·유입·페이지·시/도 (어제까지 확정 데이터 기준, 「최근 30분 활성」 참고) |

## 운영 시 참고

- 관리자에서 등록·수정·삭제하면 **공개 사이트에 바로 반영**됩니다.
- 공지·채용·실적 목록은 **등록일 최신순**으로 표시됩니다.
- 채용 **마감 표시**는 기존 마감 글만 유지되며, 관리자에서 마감 토글은 없습니다.
- 문의하기 메일은 기존과 동일하게 Resend API로 수신됩니다.

## 방문 통계 참고

- 숫자는 **어제까지 집계 완료된 데이터** 기준이며, 오늘 데이터는 1~2일 후 반영됩니다.
- **「최근 30분 활성」**은 최근 30분간 방문한 누적 수이며, 지금 동시에 보고 있는 인원수가 아닙니다.
- 관리자(`/admin`) 방문은 통계에 포함되지 않습니다.
